    Frequently Asked Questions (FAQs)

    How to Participate

    There are many ways to participate and we're always looking for contributors. Just on the ODF Toolkit Project and afterwards watch this project. Then you may want to start with reading and posting on the mailing lists or report bugs or write documentation.

    If you're a developer, just dive into the development section. Here you find information on where to access the source code and how to set up a build environment.

    How to contribute to an open-source project on GitHub

    General information on doing the first pull-request can be found on this blog.

    How to get in contact

    For any questions, comments, or concerns about this project or website please contact the project on the developer mail list.

    How to report an issue

    Please create issue on our GitHub.

    How to edit our website

    The pages on this website are written in markdown syntax and stored in our GIT repro at src/site. Detailed information can be found on our Website Development page.
